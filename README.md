## 🚀 Featured Projects

### 🤖 AI Email Agent
> FastAPI + DeepSeek-R1 (via Ollama) + Gmail API + ChromaDB

Fully autonomous email agent that **reads incoming Gmail**, **generates professional AI replies** using a locally-running LLM, and **auto-sends responses**. Features intelligent spam filtering, persistent memory with ChromaDB for context-awareness, and secure OAuth2 handling.

`FastAPI` `Ollama` `DeepSeek-R1` `ChromaDB` `Gmail API` `OAuth2`

---

### 📚 AskMyDocs — Personal Knowledge Base
> FastAPI + LangChain + ChromaDB + Ollama

Intelligent Q&A system over **user-uploaded PDFs and YouTube videos**. Uses `YoutubeLoader` for transcript extraction, `nomic-embed-text` embeddings for semantic chunking, LangChain LCEL chains for context-aware retrieval, and a clean Tailwind CSS chat interface.

`FastAPI` `LangChain` `ChromaDB` `RAG` `Tailwind CSS` `YouTube Loader`

---

### ⚖️ AI Legal Document Simplifier — "Legal to Simple"
> Django + Google Gemini AI + PyPDF + Render

Django web app that converts **complex legal PDF documents** into simple **Hinglish explanations** using Gemini AI. Supports PDF uploads with page-wise PyPDF extraction, AI-powered summarization, and production deployment via Gunicorn on Render.

`Django` `Gemini AI` `PyPDF` `Gunicorn` `Render`

---

### 🚦 Smart Traffic Violation Detection
> Python + YOLOv8 + EasyOCR + OpenCV + SQLite

Computer vision system that **detects helmet violations & triple riding** from CCTV footage. Includes automated **number plate recognition (ANPR)**, challan auto-generation, offender database, and dashboards.

`YOLOv8` `EasyOCR` `OpenCV` `SQLite` `Python`

---

### 📊 Social Media Engagement Predictor
> Python + Scikit-learn + Pandas + Matplotlib

Multiple Linear Regression model predicting **Instagram/YouTube post engagement** from features like follower count, post type, hashtags, caption length, and past engagement metrics.

`Python` `Scikit-learn` `Pandas` `Matplotlib` `Regression`
